Color Palette

A pastel color palette was chosen to align with the mascot and create a cohesive visual identity while remaining easy on the eyes during prolonged use. These softer tones help reduce visual fatigue and support a calm, approachable experience. The primary colorway pairs warm orange-yellow tones with cool blues. As complementary colors, they create visual balance and contrast while maintaining clarity and usability across the interface.

Main Hub

In the first iteration, the app used a plain background, which felt visually flat and lacked context. To better ground the mascot in the interface, I introduced a “room” environment inspired by common patterns in similar apps (e.g., Finch). The room transitions from a bright, sunlit living space to a calmer bedroom depending on the time of day, reinforcing a sense of continuity and helping the pet feel aligned with the user’s daily rhythm.

The initial navigation bar was redesigned to improve clarity and usability. While the original 3D, inflated icons matched the overall aesthetic, simpler vector icons were ultimately chosen to make navigation more immediately recognizable and easier to scan.

Components

To simplify implementation and improve maintainability, the interface was built using reusable components such as buttons and input fields. This approach ensured visual consistency across the app and aligned well with React Native’s component-based architecture.
